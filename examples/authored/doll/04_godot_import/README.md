## Regenerate

There is nothing to regenerate in this folder - the files are authored once and edited by hand. To rebuild the **inner generated scene** that this wrapper instances:

1. Open `../03_blender_setup/doll_rigged.blend` in Blender.
2. Enable the **Proscenio** addon.
3. `File > Export > Proscenio scene...` -> targets the Godot project's `res://examples/doll/`.
4. In Godot, let the importer regenerate `doll.proscenio` -> `doll.scn`.

## Wrapper script contract (SPEC 001)

The wrapper is **separate** from the imported scene so reimports never touch it. Customise gameplay logic (`autoplay` flag, animation defaults, signal hookups) in `Doll.gd`; never edit the inner generated scene. See `.ai/skills/godot-dev.md` for the full pattern.
