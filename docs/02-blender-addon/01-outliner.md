# Outliner

A sprite-centric flat list of the objects Proscenio cares about - slots, sprite meshes, attachments, and armatures - so you do not have to scroll Blender's native outliner on a big rig.

Type a substring to filter live; click a row to make that object active and selected. The SOLO icon pins a row as a favorite, and the favorites toggle next to the filter hides everything else. Slots render first with a `[slot]` prefix and their attachments indented under them; armatures render last with `[arm]`.

This panel is **blender-only** - favorites and filter state never reach the export.
