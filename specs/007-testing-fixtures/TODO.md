# SPEC 007 — TODO

Builds two new test fixtures (`simple_doll/`, `blink_eyes/`) covering the workflows the existing `dummy/` does not. Each fixture is generated by a deterministic headless-Blender build script so the underlying art is auditable from source. See [STUDY.md](STUDY.md) for the full design.

## Decision lock-in

- [x] D1 — fixtures live under `examples/` (same as `dummy/` today).
- [x] D2 — PNG sources are programmatically generated (no Photoshop dependency).
- [x] D3 — `.blend` files committed; build script kept around for re-creation.
- [x] D4 — sprite_frame layer naming convention `<name>_<index>` (locks for SPEC 006).
- [x] D5 — build scripts under `scripts/fixtures/`.
- [x] D6 — build scripts use headless Blender.
- [x] D7 — one CI job iterates every fixture.
- [x] D8 — `dummy/` stays as the shared-atlas / sliced-repack stress test.

## simple_doll fixture

- [ ] `scripts/fixtures/build_simple_doll.py` — headless build:
  - generates `examples/simple_doll/layers/*.png` (Pillow or numpy + bpy.types.Image)
  - assembles `simple_doll.blend` from scratch (5 polygon meshes + 6-bone armature + weights + 2 actions)
  - re-runnable: deletes existing outputs and rewrites them deterministically
- [ ] `examples/simple_doll/layers/*.png` checked in (small, deterministic)
- [ ] `examples/simple_doll/simple_doll.blend` checked in (LFS)
- [ ] `examples/simple_doll/simple_doll.expected.proscenio` golden — CI diffs against re-export
- [ ] `examples/simple_doll/Doll.tscn` + `Doll.gd` — Godot wrapper following SPEC 001 pattern
- [ ] `examples/simple_doll/README.md` — what this fixture tests, what fails when broken

## blink_eyes fixture

- [ ] `scripts/fixtures/build_blink_eyes.py` — headless build:
  - generates `examples/blink_eyes/layers/eye_0.png` … `eye_3.png` (32x32 each, 4 frames of a blink)
  - assembles `blink_eyes.blend`: 1 sprite_frame mesh + 1-bone armature + 1 action animating `frame` 0→1→2→3→2→1→0 over 12 frames
  - sprite_frame metadata: `hframes=4`, `vframes=1`
- [ ] `examples/blink_eyes/layers/*.png` checked in
- [ ] `examples/blink_eyes/blink_eyes.blend` checked in (LFS)
- [ ] `examples/blink_eyes/blink_eyes.expected.proscenio` golden
- [ ] `examples/blink_eyes/BlinkEyes.tscn` + `BlinkEyes.gd` — Godot wrapper
- [ ] `examples/blink_eyes/README.md`
